# Apex - Galaxy

**Step 1: Pre Injection**

1. Enable GeForce Experience IN-GAME overlay for NVIDIA GPU or[ SteelSeries GG Sonar](https://steelseries.com/gg) overlay for AMD GPU
2. Make Sure you followed all step on the Getting started page!

**Step 2: Configure Settings (Steelseries GG)**

1. Go to settings
2. Navigate to SONAR → Shortcuts tab
3. Find "Master - Volume Up"
4. Set it to F7 key
5. This bind must be pressed before injection

**Step 3: Using the loader**

1. Download the loader[ here](https://undetect.net/loader_exodus)​
2. Place the loader on your Desktop
3. Run the loader as Admin
4. The first time you use the loader it might restart your pc
5. Enter your license key by pressing <mark style="color:red;">"Right Mouse Button"</mark> and click the <mark style="color:red;">"Enter"</mark> button on your keyboard
6. Launch the game once CMD is closed
7. Open Menu with "Insert" (Can also use On-Screen Keyboard)
