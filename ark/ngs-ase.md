# ARK - NGS ASE

**Step 1: Register 🖥️​**

1. Use Google Chrome
2. Register page[ Here](https://papa.rip)​
3. Register / Login
4. Go to subscriptions and press activate Key
5. Enter Key and press activate
6. Go to Loader page and Press Download

**Step 2: Injection**

1. Open the Loader as Admin

{% hint style="danger" %}
Loader should auto close and delete itself
{% endhint %}

2. Go back to the Loader page and wait for ark to load there\
   (Should like like this below)

{% hint style="danger" %}
Could take 1/5 Minutes)
{% endhint %}

\
