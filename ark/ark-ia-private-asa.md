# ARK - IA ASA / ASE

**Step 1: Register 🖥️​**

1. Register Page[ Here](https://intelligent-aiming.xyz/forum/index.php?register)​
2. Register / Login

**Step 3: Activate License Key**

1. Activation Page[ Here](https://intelligent-aiming.xyz/redeem-key)​
2. Enter License Key
3. Press Redeem key

**Step 3: Download Loader**

1. Download Loader [Here](https://intelligent-aiming.xyz/forum/index.php?threads/loader-download.916/)
2. Unzip the Loader\
   (password is "RUST" Is also displayed on the download page)
3. Close all background apps
4. Run Loader as Admin
5. Login to your account
6. Press the selected Game version
7. Press Load
8. Wait for the beep\
   (Loader should close itself)
9. Launch Ark
10. Open Menu with "Insert" (Can also use On-Screen Keyboard)
