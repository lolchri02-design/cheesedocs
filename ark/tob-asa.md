# ARK - TOB ASA

**Step 1: Register 🖥️​**

1. Use Google Chrome
2. Register page[ Here](https://tob.wtf/)​
3. Register / Login
4. Go to subscriptions and press activate Key
5. Enter Key and press activate
6. Go to Loader page and Press Download

**Step 2: Injection**

1. Open the Loader as Admin\
   (Loader should auto close and delete itself)
2. Go back to the Loader page and wait for ark to load there (Could take 1/5 Minutes)\
   (Should like like this below)\
   ![](https://2395209493-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Ftr5o4rmuHoCVCvn6o3Tj%2Fuploads%2FWV8WtSsVVBbRRaS7EgfC%2Fimage.png?alt=media\&token=0717201d-12c9-405c-b1d8-d91eed822545)
3. Whenever it loaded press the play icon on the left side, as shown below.\
   ![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXcL0Yx--bTuC9c5-kWyjgWW9l2QiLxpErnnAGbUqJorNyrwNr5owMG6NKYK1RBa1QBQTKwirULHeG1YkuHEzw3dL6QyveT5piREU8KeyH-SNOsWZ1T89mz3PTf22anHhQOCB8Pw?key=BJhMjbdmj49g54Wua29TNtFM)
4. ​Wait until you get a popup saying Success.
5. Launch Ark
6. When in Main Menu Press "F8"
7. TOB Watermark should appear in the top left
8. Go in a server and press F8 for menu to show\
   (Menu doesn't show in main menu)
