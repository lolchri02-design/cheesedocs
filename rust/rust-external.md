# Rust - External

**Step 1: Loader Pre Setup**

1. Disable any antivirus software, windows real time protection, and any running anticheats (faceit, vanguard, etc)
2. Go into bios and disable secure boot and set bios mode to UEFI
3. Go to[ https://lewislitt.life/Store/install.html](https://lewislitt.life/Store/install.html) and enter a key and wait for a custom loader build to be downloaded

**Step 2: Login and Injection**

1. Launch loader as admin and either login to an existing account or register a new one using a license
2. Head over to the products page and select the product you would like to use and press inject
3. If loader says “Please Launch SteelSeriesGG”, download SteelSeries and run it ([Click here for download](https://steelseries.com/gg/moments/download))
