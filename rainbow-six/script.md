# R6 - Script

**Step 1: Injection**

1. FIRST MAKE SURE GAME IS NOT RUNNING IN YOUR TASK MANAGER
2. Download the Loader [HERE](https://www.dropbox.com/scl/fi/w48h6chm1h7lpc9hoj0by/RecoilV2.exe?rlkey=wi8kk6m00pftmjt4435gh1vou\&st=7ux45q7h\&dl=1)
3. Run the loader as admin
4. Make an account if it's the first time, otherwise log in with your credentials
5. Select your subscription
6. Press the load button
7. After the menu opens you can open the game

**Step 2:** How to get help with recoil

1. First put your sensitivities in the slider
2. Select your side , attack or defense
3. Select your operator
4. Select the gun you are using , and the secondary if its the case
5. Please input the same attachments that you are using in game ( You can also save them with the button for the future , it is racomended )
6. Simply press the activate button and you are done !

*Keep in mind, using the hotkeys to switch from secondary to first and to activate and deactivate the recoil is very important and can make the experience easier!*
