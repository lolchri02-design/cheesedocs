# FiveM - Macho

**Step 1: First injection**

* Download the loader[ here](https://machocheats.com/web/download/)
* Put the loader in a folder somewhere on a drive (Not on the desktop)
* Run the loader as Admin
* Enter your key
* Select the product you wanna load
* Press load product
* Wait for the loader to say injected
* Launch the game (Windowed or Borderless)

Open Menu with "Insert" (Can also use On-Screen Keyboard)
