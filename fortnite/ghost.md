**Step 2: Injection**

1. Install and run the [Loader](https://mega.nz/folder/I21UGKpI#wiZrUW-hgkt_G3ay7pZ8NA) as admin
2. First launch it might restart your pc
3. Run Loader as admin and enter your key
4. Wait untill succes message / loader closes
5. Launch game and press f2 in main menu\
   (most of the time menu already opens, then no need to press f2)

* Make Sure your game is in borderless / windowed fullscreen
* "Insert" to Open / Close Menu!
