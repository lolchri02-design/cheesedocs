# War Thunder - Labcore

**Step 1: Pre injection**

1. Open Discord and go to user settings
2. Scroll down to Game-Overlay
3. Activate Overlay

**Step 2: Using the loader and software**

1. Download the loader[ here](https://panel.labcorecheats.com/loader)
2. If its your first time using a LabCore product make a account
3. Extract your loader
4. Make sure the loader isnt running from your Desktop (Place it on a drive in a new folder)
5. Run the loader as Admin
6. Copy paste you key into the loader
7. If the key is entered correctly, the loader will display status : Active in the bottom left corner.
8. Once the bootstrap indicates success or closes, proceed to open War Thunder
9. Press the F1 key once in the main menu to initialize the software

* Open Menu with "Insert" (Can also use On-Screen Keyboard)
